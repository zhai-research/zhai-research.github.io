## 1. 研究问题

浮式海上风力机 (FOWT) 的运动同时受到波浪和风湍流的耦合激励，能否利用 FOWT 自身的运动传感器数据，像"波浪浮标"一样，同时反演出方向波谱和风湍流谱？这可以替代专用海洋气象仪器，降低成本。

<!-- more -->

## 2. 研究空缺

- 传统 WBA (Wave Buoy Analogy) 主要用于船舶/FPSO 等水动力主导平台，无法直接应用于 FOWT（气动-水动耦合强）
- 风力机转子产生的非定常气动力会"污染"运动响应，标准 WBA 会将气动贡献误判为波浪能量
- 尚无研究同时从 FOWT 运动数据中恢复方向波谱和风湍流谱

## 3. 核心思想

用 1D U-Net 处理 FOWT 的 6-DOF 运动和塔顶加速度的频谱特征，联合输出：(1) 二维方向波谱 S_dir(f,θ)，(2) 三个风速分量的功率谱密度 S_u/v/w(f)。通过多任务学习，利用塔顶加速度分离气动和水动响应。

## 4. 方法结构

**输入特征**：9个传感器（6-DOF运动 + 3个塔顶加速度）→ FFT → 幅度谱 + 相位谱 + 36对交叉谱 → 200×90 矩阵

**网络架构**：共享编码器（3个1D卷积块） + 瓶颈层 + 两个任务头
- 波浪头：对称解码器（转置卷积 + skip connection）→ 200×72 方向波谱
- 风头：全局平均池化 + MLP → 512×3 风湍流 PSD

**训练**：OpenFAST 仿真 NREL 5-MW OC3-Hywind，5000个海况，韩国蔚山外海，Adam优化，lr=0.001

**关键指标**：Hs R²=0.98, Tp R²=0.99, Dm MAE=2.19°, 风速 R²=0.96

## 5. 主要贡献

1. 首次实现从 FOWT 运动数据联合反演方向波谱和风湍流谱
2. 1D U-Net 架构分离气动-水动耦合响应
3. 消融实验确认塔顶加速度和相位信息的关键作用
4. 鲁棒性验证：严重噪声和数据缺失下仍保持精度
5. 迁移学习验证可适应不同频谱模型（Pierson-Moskowitz, IEC Class C）

## 6. 实验设计与结果

**数据**：OpenFAST 仿真，5000海况，30min/海况，20Hz采样
**划分**：70/15/15

**方向波谱结果**：
- Hs: MAPE 2.8%, R²=0.98
- Tp: MAPE 1.1%, R²=0.99
- Dm: MAE 2.19°
- Dp: MAE 2.30°

**风谱结果**：
- 风速: R²=0.96
- 湍流动能: R²=0.91
- 湍流强度: R²=0.84

**与 MLP 基线对比**：U-Net 在低能海况和大风浪夹角下显著优于 MLP
- 低能海况 MLP Hs MAPE >20%，U-Net <10%
- 风浪夹角 118° 时 MLP 退化严重，U-Net 仍保持精度

**消融关键发现**：
- 移除塔顶加速度 → Hs MAPE 从 2.8% 升至 6.5%（误差翻倍以上）
- 移除相位 + 交叉谱 → 方向预测几乎完全失效（Dm MAE >70°）
- 单任务 vs 多任务 → 波浪预测精度几乎不变，多任务不损害主任务

**输入时长**：10min 为平衡点（Hs MAPE 4.8%），30min 最优（2.8%）

**鲁棒性**：
- 训练时加入噪声/偏置/缺失扰动的模型，在严重退化（SNR=10dB, 20%缺失）下误差比清洁训练模型低 45%

**局限案例**：
- 停机状态（U>25m/s）：训练数据极少，风速低估、波高高估
- 双峰海况：训练集无双峰谱，无法恢复副峰

## 7. 方法优缺点

**优势**：
- 无需专用海洋气象仪器，利用 FOWT 自带传感器
- 同时反演波谱和风谱，超越传统 WBA
- U-Net 多尺度结构保留频谱连续性
- 对传感器噪声和数据缺失鲁棒

**局限**：
- 训练数据基于 JONSWAP + IEC Kaimal 参数化谱，非参数谱泛化需迁移学习
- 仅在 OC3-Hywind spar 平台上验证
- 停机状态和双峰海况训练数据不足
- 依赖 20Hz 同步高频采样，标准 SCADA 通常只有 10min 均值

## 8. 可以如何改进

1. 扩展训练集覆盖停机状态和双峰海况
2. 验证其他平台类型（半潜式、TLP）
3. 现场数据验证（非仅仿真）
4. 降低采样率需求以适配标准 SCADA
5. 与波高重建方法结合：FOWT 作为移动观测站补充浮标网络

## 9. 用 3 句话总结论文

第 1 句：本文首次解决从浮式风机运动数据联合反演方向波谱和风湍流谱的开放问题。  
第 2 句：1D U-Net 模型对波浪参数 R²>0.98、风速 R²=0.96，在传感器噪声和数据缺失下保持鲁棒。  
第 3 句：FOWT 作为"海洋气象传感器"的思路有创新性，可为波浪观测网络补充稀疏海域数据。
